01
Present Simple vs Present Continuous
✓
▶

Present Simple — привычки, факты, расписания. Маркеры: always, usually, every day, never.
Present Continuous — действие прямо сейчас или временное. Маркеры: now, at the moment, currently, right now.

ВремяФормаПримерПеревод
Present SimpleI/You/We/They + V
He/She/It + V+s		I write code every day.
She deploys the app on Fridays.		Я пишу код каждый день.
Она деплоит по пятницам.
Present Continuousam/is/are + V-ingI am writing a function now.
The server is loading data.		Я пишу функцию сейчас.
Сервер загружает данные.
Present Simple (факт/привычка) My AI assistant usually processes voice commands. → Мой ИИ-ассистент обычно обрабатывает голосовые команды. Python uses indentation instead of curly braces. → Python использует отступы вместо фигурных скобок. Present Continuous (прямо сейчас) Right now it is connecting to the Gemini API. → Прямо сейчас он подключается к Gemini API. I am currently learning backend development. → В данный момент я изучаю backend разработку.
⚠ Частая ошибка

НЕ используй Continuous для глаголов состояния: know, understand, believe, want, need, love, hate. ✗ I am knowing Python. → ✓ I know Python.

Задание дня

Напиши 10 предложений о своём проекте (Lux или любом другом). 5 — Simple (что делает обычно), 5 — Continuous (что делает прямо сейчас). Используй связку "To be honest" и "However" хотя бы по одному разу.

Показать пример ответа

My assistant usually processes voice commands and connects to external APIs. It always remembers my preferences. However, right now it is searching for relevant data. To be honest, it is getting smarter every day. It currently is analysing my coding patterns.

Проверь: нет ли Continuous с глаголами know/want/need? Есть ли оба маркера "To be honest" и "However"?

02
Shadow Cinema: произношение и Linking Sounds
✓
▶

Shadowing (теневое повторение) — техника при которой ты повторяешь речь носителя одновременно с ним, как «тень». Лучший метод для постановки произношения и интонации. Используют полиглоты и дипломаты.

Согласный + Гласный (самое частое) "pick it up" → звучит как "pick-i-tup" "turn it on" → звучит как "tur-ni-ton" "log in and out" → звучит как "lo-gi-nan-dout" Конечный согласный + начальный согласный "best code" → t почти исчезает: "bes-code" "back-end developer" → "backen-developer" Гласный + гласный (вставляется W или Y) "go online" → вставляется W: "go-w-online" "I asked" → вставляется Y: "I-y-asked"
СловоПереводПример
broadcastтрансляция, вещаниеThe event was broadcast live.
compellingзахватывающий, убедительныйA compelling narrative keeps viewers hooked.
coverageосвещение (в СМИ)Media coverage of the event was massive.
grippingзахватывающийThe plot twists were gripping.
editorialредакционный, передовицаThe editorial team approved the story.

  2. Vocabulary: Напиши мини-рецензию (30–40 слов) на любой сериал («Wednesday», «Mr. Robot», «You»), используя минимум 4 слова из таблицы выше.
Пример рецензии

✓
▶

Используется для завершённых действий в прошлом. Regular verbs: +ed. Irregular: learn by heart (buy→bought, write→wrote, have→had).

InfinitivePast SimpleПереводПример
writewroteписатьI wrote a script yesterday.
buildbuiltстроитьShe built the API last week.

runranзапускатьHe ran the tests.
breakbrokeломатьThe update broke production.
sendsentотправлятьI sent the PR for review.
understandunderstoodпониматьI finally understood recursion.
⚠ Классическая ошибка

НЕ используй "have" для завершённых действий с указанием времени. ✗ I have ordered it yesterday. → ✓ I ordered it yesterday.

Тема: "Is technology making us lonely?" Структура Introduction: [Перефразировать тему] + [Своя позиция] + [Что будет в эссе] It is often argued that digital technology connects people across the globe. However, in my view, the rise of social media and constant online communication has actually contributed to increased feelings of isolation in modern society. This essay will explore both perspectives before presenting my conclusion. → Обрати внимание: не повторяй слова из темы дословно — перефразируй!
Задание дня
  1. Grammar: Опиши свою последнюю покупку на Pinduoduo (или любую другую). Что заказал? Сколько стоило? Когда пришло? Только Past Simple, 5–6 предложений.
  2. IELTS Writing: Напиши только Introduction (2–3 предложения) на тему: "Social media does more harm than good."
Пример ответа

I ordered a mechanical keyboard on Pinduoduo last week. It cost around 8,000 tenge. The delivery took 12 days and arrived on Monday. The quality was better than I expected. I immediately started using it for my coding sessions.

IELTS intro: While social media platforms have revolutionised communication worldwide, I strongly believe their negative effects outweigh the benefits for society. This essay will examine how they contribute to mental health issues and misinformation.

✓
▶

Present PerfectPast Simple
I have built three Python projects. (опыт)I built a project last month. (конкретное время)
She has never used Docker. (опыт)She never used Docker in her last job.
We have already deployed it. (результат)We deployed it on Monday.

✓
▶
Формула: am/is/are/was/were + Past Participle Active: The developer stores data in the database. Passive: Data is stored in the database. Active: We hash passwords using bcrypt. Passive: Passwords are hashed using bcrypt. Active: The team deployed the app yesterday. Passive: The app was deployed yesterday. → В технической документации Passive используется постоянно. We store API keys safely.API keys are stored in environment variables.

IELTS Writing Task 2 — Полное эссе
✓
▶
Paragraph 1 — Introduction (2–3 предложения) Перефразируй тему + своя позиция + что будешь обсуждать Paragraph 2 — Body 1 (основной аргумент) Topic sentence → Explanation → Example → Result Paragraph 3 — Body 2 (контраргумент или второй аргумент) However, some argue that... → Explanation → Concession Paragraph 4 — Conclusion (2–3 предложения) In conclusion... / To summarise... → повтори позицию другими словами
Introduction: The rapid advancement of artificial intelligence has sparked debate about its impact on employment. While I acknowledge that automation will eliminate certain roles, I firmly believe that AI will ultimately create more opportunities than it destroys. Body 1: Firstly, history shows that technological revolutions consistently generate new industries. The internet, for instance, eliminated travel agencies but created millions of jobs in e-commerce and digital marketing. Similarly, AI is already creating roles in machine learning engineering, data science, and AI ethics. Body 2: However, the transition will be painful for many workers. Low-skilled and repetitive jobs are particularly vulnerable. Governments must invest in retraining programmes to help displaced workers adapt to the changing economy. Conclusion: In conclusion, while AI will undoubtedly transform the job market, I believe it represents an opportunity rather than a threat, provided we manage the transition responsibly.
⏱ IELTS совет

40 минут: 5 мин — план, 30 мин — написание, 5 мин — проверка. Никогда не пропускай Conclusion. Aim for Band 7: разнообразная лексика, сложные структуры, чёткая позиция.

Задание дня

Напиши полное эссе (250 слов) на тему: "Technology has made people less patient. To what extent do you agree?". Следуй структуре. Используй минимум 3 linking words (Furthermore, However, In conclusion, Moreover, Nevertheless).

💡 Зачем collocations?

Native speakers не говорят "make a mistake" — они говорят именно так. Collocations делают речь естественной. Ошибка: "do a mistake" ✗, "make a mistake" ✓. Учи слова не по отдельности, а вместе с их партнёрами.
